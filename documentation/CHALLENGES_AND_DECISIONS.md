## Classification Guide

- `Resolved`: fix delivered and verified.
- `Monitoring`: fix delivered, still being observed.
- `Open`: unresolved and actively impacting progress.
- `Accepted Risk`: known issue intentionally deferred.

## Entry Rule

Every non-trivial blocker should be added with:

1. Clear root cause summary.
2. Concrete technical resolution.
3. Named owner and explicit next step.
